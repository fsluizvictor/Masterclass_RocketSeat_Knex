# Masterclass_Knex
O projeto consiste na execução de uma API que irá ter CRUD de usuários e projetos desses usuários.

## Tecnologias
* Node JS
* Express
* Knex

## Linguagem e Banco de Dados
* JavaScript e PostgreSQL

## Funcionalidades desenvolvidas
* CRUD
* Paginação de Projetos
* Soft Delete (user)
* Procedures & Triggers
